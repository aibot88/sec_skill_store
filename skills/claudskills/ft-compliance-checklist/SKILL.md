# 合规检查清单生成器

按行业+企业规模+纳税人类型自动生成结构化合规检查清单。生成清单后交给 ft-compliance-auditor 执行检查。本技能只负责"列出该查什么"，不负责"实际去查"。

## Core Knowledge

### 月度检查项 (每月必查)

按时间窗口(代账41任务模型):

**采集期 (1-5日)**
- [ ] 银行对账单下载/导入
- [ ] 费用报销单据收集
- [ ] 进项发票归集确认
- [ ] 销项发票开具情况核对
- [ ] 社保/公积金缴费通知确认

**征期 (6-15日)**
- [ ] 增值税申报(一般纳税人月度/小规模季度)
- [ ] 个人所得税代扣代缴申报
- [ ] 附加税(城建税/教育费附加/地方教育费附加)申报
- [ ] 印花税申报(按次/按季，看当地规定)
- [ ] 社保/公积金缴费

**质检期 (16-24日)**
- [ ] 银行余额调节表编制
- [ ] 往来账核对(应收/应付/预收/预付)
- [ ] 费用明细检查(招待费/差旅费/办公费分类)
- [ ] 进项抵扣台账更新
- [ ] 收入成本配比检查

**准备期 (25-31日)**
- [ ] 下月申报数据预审
- [ ] 固定资产折旧计提
- [ ] 工资薪金计提
- [ ] 税金及附加计提
- [ ] 期末结转(收入/成本/费用)

### 季度检查项 (额外)

- [ ] 企业所得税预缴申报(A类/B类)
- [ ] 财务报表报送(资产负债表+利润表)
- [ ] 增值税季度申报(小规模纳税人)
- [ ] 印花税季度汇总申报(部分地区)
- [ ] 关联交易台账更新(如有关联方)
- [ ] 税负率自查(增值税税负/所得税税负与行业均值比对)

### 年度检查项

- [ ] 企业所得税汇算清缴(次年1月1日-5月31日)
- [ ] 个人所得税汇算清缴(次年3月1日-6月30日)
- [ ] 工商年报(次年1月1日-6月30日)
- [ ] 社保缴费基数年度调整(一般7月)
- [ ] 残疾人就业保障金申报
- [ ] 工会经费申报
- [ ] 房产税/城镇土地使用税(按半年或按年)
- [ ] 关联交易年度报告(关联交易金额超标的)
- [ ] 资产损失税前扣除申报(如有)
- [ ] 税收优惠备案/留存备查(高新/小微/研发加计等)

### 行业特殊检查项

| 行业 | 额外检查项 |
|---|---|
| 建筑业 | 项目预缴增值税(一般计税2%/简易计税3%)、异地经营报验、完工百分比确认 |
| 房地产 | 土地增值税预缴/清算、预售收入确认、开发成本分摊 |
| 外贸/进出口 | 出口退税申报(收齐凭证次月申报期内，2022年起放宽)、外汇核销、海关缴款书抵扣 |
| 餐饮/零售 | 农产品抵扣台账、库存盘点、现金收入完整性 |
| 高新技术企业 | 研发费用辅助账(按项目归集)、高新资质年度自查、知识产权台账 |
| 电商 | 平台收入确认(GMV vs 实际收入)、刷单退款冲减、平台服务费进项 |
| 物流/运输 | 油费进项台账、通行费发票归集、车辆保险/折旧 |

### 企业规模差异

| 类型 | 典型差异 |
|---|---|
| 小规模纳税人 | 增值税按季申报、免征额(月销售额10万/季销售额30万，仅限普票部分)、不需要进项认证 |
| 一般纳税人 | 增值税按月申报、进项认证勾选、发票管理更严格 |
| 小微企业 | 所得税优惠(应纳税所得额<=300万，实际税负约5%=25%x20%，需确认当年减半政策是否延续)、六税两费减半 |
| 高新技术企业 | 所得税15%优惠税率、研发加计扣除100%(2023年起所有企业统一，实际发生额的200%税前扣除)、额外季度台账 |

## Decision Framework

| 用户需求 | 生成哪种清单 | 关键参数 |
|---|---|---|
| "这个月要做什么" | 月度清单(当前时间窗口) | 纳税人类型+行业 |
| "季度末要注意什么" | 季度清单(含月度) | 同上+是否有关联方 |
| "年底要准备什么" | 年度清单(含所有) | 同上+是否高新/小微+是否有资产处置 |
| "建筑公司要查什么" | 行业定制清单 | 行业特殊项叠加通用项 |
| "新接的客户第一个月" | 首月清单(含建账) | 接手时的特殊检查项(历史数据移交/期初余额确认) |

## Gotchas

1. **清单生成不等于清单执行** -- 本技能只生成"该查什么"的清单。实际执行检查(打开系统、核对数据、标记问题)是 ft-compliance-auditor 的工作。不要在清单生成阶段就开始做判断或给结论。

2. **不同地区征期可能不同** -- 印花税在上海是按季申报，在部分地区是按次申报。房产税/城镇土地使用税的申报频率各地也不统一。生成清单时必须确认客户所在地区的具体征期安排，不能一刀切用全国标准。

3. **高新技术企业有额外的季度研发台账检查项** -- 高新企业每季度需要更新研发费用辅助账(按项目归集人工/材料/折旧/其他费用)。这个检查项很多代账公司会漏，直到年审或税务检查时才发现台账没做。

4. **小微企业和一般企业的检查项差30%以上** -- 小微企业享受大量简化和优惠(六税两费减半、所得税优惠、增值税免征)，对应的检查项更少但判定条件更多(要持续监控是否还符合小微标准)。不能把一般企业的完整清单直接套用。

5. **首月接手客户的清单最复杂** -- 新接客户第一个月需要额外检查：期初余额确认、历史账务移交完整性、税种鉴定核对、网上申报开通状态、发票领购状态、社保账户状态。这些一次性检查项容易遗漏。

6. **汇算清缴季(1-5月)清单需要提前3个月准备** -- 很多代账公司12月才开始想汇算清缴的事。实际上10月就应该开始：资产盘点、坏账确认、关联交易整理、研发费用归集复核。清单应在10月就生成汇算清缴准备项。

## When to Escalate

- 客户业务涉及多个行业(如既有建筑又有房地产)：行业清单叠加可能产生冲突，需要 ft-business-analyst 先理清业务架构
- 客户有境外业务或境外投资：需要增加跨境税务检查项，超出标准清单范围
- 客户正在被税务稽查：需要生成稽查应对专项清单，这不是常规合规清单
- 客户要求出具正式的内控合规报告：需要注册会计师签字，超出代账范围

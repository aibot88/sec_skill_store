# Skill 功能清单

## skill技能管理
*   **`find-skills`**: 在海量技能库中搜索，快速找到你需要的Skill
*   **`skill-creator`**: 创建自己的专属技能，定制个性化工作流
*   **`skill-vetter`**: 安全审查工具，安装任何社区Skill前先用它扫一遍，防恶意代码
*   **`model-usage`**: 按模型追踪token消耗和费用，避免账单惊喜
*   **`free-ride`**: 通过OpenRouter免费使用多种AI模型，省钱利器

## 文案营销类
*   **`copywriting`**: 文案全能手，优化表达、润色文字，让文案更有感染力
*   **`social-content`**: 社媒文案生成，适配小红书、朋友圈、微博等平台风格
*   **`product-marketing`**: 梳理产品卖点与用户痛点，输出清晰的营销策略

## 设计体验
*   **`ui-ux-pro-max`**: 更专业的交互与体验设计，适合产品原型和界面优化
*   **`nano-banana-pro`**: 用Gemini 3 Pro生成和编辑图片，秒出博客封面、插图等
*   **`frontend-design`**: 生成生产级前端界面和组件
*   **`diagram-generator`**: 自然语言生成流程图、架构图、思维导图

## 文档办公
*   **`pdf`**: 提取PDF内容、翻译、拆解分析
*   **`docx`**: 生成带格式的Word文档，支持标题、列表、表格等样式
*   **`xlsx`**: 生成Excel表格，支持数据分析和图表可视化
*   **`pptx`**: 直接生成PPT幻灯片，快速搞定演示文稿
*   **`Gog`**: 高效管理Gmail、日历、云端硬盘、通讯录、表格和文档等各项服务
*   **`Humanizer`**: 消除文本中人工智能生成的痕迹，使其更自然流畅

## 语音多媒体
*   **`openai-whisper`**: 本地语音转文字，无需API Key，音频不离开电脑，隐私安全，~11.5K下载
*   **`youtube-watcher`**: 抓取YouTube视频字幕/转录，配合summarize秒懂2小时长视频，~9K下载
*   **`podcast-generation`**: 用Azure OpenAI Realtime API从文本生成真实音频叙述，支持播客、有声书
*   **`voice-ai-tts`**: 文字转语音，支持多角色、多语言、语速调节，适合配音和旁白制作
*   **`podcast-producer-agent`**: 端到端播客制作：脚本生成→多角色TTS→背景音乐→音频混合，一条龙
*   **`spotify`**: 控制Spotify播放，“放点lo-fi让我工作”，让Agent更像真助手

## 通讯与协作
*   **`Slack`**: 团队消息读写、频道管理

## 知识管理与笔记
*   **`obsidian`**: 本地 Markdown 笔记管理，搜索 Vault、创建笔记、建立双链，隐私优先
*   **`apple-notes`**: 直连 Apple 备忘录，“把这段保存为笔记”
*   **`Notion`**: 帮助用户轻松创建、管理 Notion 页面、数据库和内容块，实现工作流程自动化
*   **`self-improving-agent`**: 记录错误、学习修正、积累经验库，越用越聪明

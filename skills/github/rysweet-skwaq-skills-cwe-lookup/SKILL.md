# CWE Lookup

Look up and explain CWE-$ARGUMENTS:

1. Search the knowledge base: `skwaq kb search $0`
2. Explain what this weakness category means
3. Provide real-world examples of this vulnerability type
4. List common detection methods
5. Suggest prevention strategies

If the CWE isn't in the local database, explain it from general security knowledge.

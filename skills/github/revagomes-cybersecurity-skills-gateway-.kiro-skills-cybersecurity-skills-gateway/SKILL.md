# Cybersecurity Skills Gateway

## Source Repository
https://github.com/mukul975/Anthropic-Cybersecurity-Skills

## How to Use

When asked to perform any security task:

1. **Find the right skill** — fetch the index to search by name, description, domain, or tags:
   ```
   https://raw.githubusercontent.com/mukul975/Anthropic-Cybersecurity-Skills/main/index.json
   ```

2. **Fetch the skill on demand** — given a skill `name` from the index (e.g. `performing-memory-forensics-with-volatility3`):
   ```
   https://raw.githubusercontent.com/mukul975/Anthropic-Cybersecurity-Skills/main/skills/<skill-name>/SKILL.md
   ```

3. **Follow the skill** — execute its `Workflow` section step-by-step and validate using its `Verification` section.
